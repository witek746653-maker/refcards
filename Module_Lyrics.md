### 1. ВЫБОР КОНТЕНТА [ОБЯЗАТЕЛЬНО]

- **80/20:** Разбирай те 20% фраз и образов, которые несут 80% смысла. Не объясняй каждое простое слово — только то, где новичок споткнётся.
- **Язык карточки:** Всегда указывай язык оригинала (`{{LANG_ORIGINAL}}`). Перевод и объяснения — на русском.
- **Три слоя перевода** — для каждой сложной фразы:
  - `Дословно` → что написано буквально (даже если звучит странно)
  - `Смысловой` → что это значит на самом деле, как бы сказали по-русски
  - `Почему так` → зачем автор выбрал именно эти слова (ритм, образ, культурный код)
- **Разговорный vs литературный:** Всегда отмечай — это нормальная речь или поэтический оборот, который в жизни так не говорят.
- **Запомнить vs понять:** Блоки для заучивания строятся на смысловых якорях, не на механическом повторении.
- **Акцентное выделение:** в тексте всех блоков — жирным ключевые слова, имена, цифры, смысловые пики (2–3 на абзац, не больше). Стиль журнальной вёрстки: читатель должен понять суть, пробежав только по жирному. Внутри code-блоков и ASCII-деревьев — не применять.

Для `{{GOAL}}` укажи цель:

**5.2. Полный текст с параллельным переводом [ОБЯЗАТЕЛЬНО]**

Формат — Markdown-таблица без заголовка, две колонки:

| | |
| :--- | :--- |
| [Оригинальная строка на языке оригинала] | [Смысловой перевод на русском] |
| [Следующая строка оригинала] | [Перевод следующей строки] |

Правила:
- Заголовок таблицы — пустой (две пустые ячейки `| | |`)
- Перевод — смысловой, не дословный
- **Куплеты** — открытая таблица без callout; метка над ней: `[Куплет N — название]`
- **Рефрен, бридж, финал** — оборачивай в callout `> [!quote] Рефрен` / `> [!quote] Финал`; строки таблицы с префиксом `> ` (лёгкий цвет отделяет структурные блоки, куплет остаётся главным открытым текстом)

### 7. БЛОК {{EXAMPLES_CONTENT}} (для заучивания) [ОБЯЗАТЕЛЬНО]

Ровно 3 блока. Цель — не анализ, а помощь запомнить.

`🧩 Смысловые куски` — раздели текст на 3–4 смысловых блока (не по куплетам, а по мысли). Для каждого — одна строка о чём он. Это даёт крюки памяти.

`🔑 Якорные строки` — 3–5 строк, которые легче всего запомнить первыми (припев, повтор, самая яркая метафора). Объясни почему именно они — якоря.

`🔄 Цепочка образов` — как образы в тексте связаны между собой. Нарисуй логику: [образ 1] → [образ 2] → [финальный образ]. Когда видишь связь — текст запоминается как история, не как набор строк.

### 8. БЛОК {{QUIZ_CONTENT}} [ОБЯЗАТЕЛЬНО]

Ровно 3 вопроса. Цель — проверить что текст понят, а не выучен механически.

- `❓ [Переведи своими словами]` — дай строку, попроси объяснить смысл так, как объяснил бы другу — не переводить дословно.
- `❓ [Найди образ]` — опиши ситуацию из жизни, попроси найти строчку из текста которая про это.
- `❓ [Восстанови логику]` — дай финальную строку / образ, попроси объяснить как автор к нему пришёл — что шло до.
