# Спецификация идеальной карточки (Layout Blueprint)

Этот документ — эталон расположения элементов. Если верстка «поплывет», восстанавливать по этим правилам.

## 1. Формат и Сетка (Grid)
- **Лист**: A4 (210×297мм).
- **Поля**: 10–15мм со всех сторон.
- **Колонки**: 2 колонки (стандарт). 
- **Межколоночный интервал (Gap)**: 8мм.
- **Выравнивание**: По ширине (justify) с автоматическими переносами (hyphens).
- **Заполнение колонок**: Последовательное (`column-fill: auto`), чтобы избежать пустот снизу.
- **Группировка**: H3 и первый блок текста не должны разрываться (`break-after: avoid` на заголовке). Сама секция может переноситься между колонками.

## 2. Иерархия заголовков
- **H1 (Заголовок всей карты)**: 
  - На всю ширину страницы (`column-span: all`).
  - Нижняя граница (линия) обязательна.
- **H2 (Крупные смысловые блоки)**: 
  - На всю ширину страницы (`column-span: all`).
  - Цветной фон, инверсный текст.
- **H3 (Подзаголовки внутри тем)**: 
  - Строго внутри одной колонки.
  - Левая цветная граница для быстрого сканирования взглядом.
- **H4**: Малый заголовок внутри текста, жирный, цвет темы.

## 3. Спец-блоки (Magazine Layout)
- **Header Info (Шапка)**: Двухколоночный блок после H1 (`.header-info`). На всю ширину (`column-span: all`). Слева — метаданные, справа — таблица статусов.
- **Summary (Итог)**: Самый последний блок в файле. На всю ширину, рамка, цветной фон.
- **Код (Code blocks)**: Только внутри колонок. Шрифт моноширинный. Перенос длинных строк обязателен.
- **Таблицы**: 
  - 2–3 колонки — внутри одной колонки страницы.
  - 4+ колонки — растягиваются на всю ширину страницы (`column-span: all`).

## 4. Правила «Чистой страницы»
- **Разрывы**: Запрещено оставлять заголовок внизу колонки без текста (break-after: avoid).
- **Группировка**: H3 и следующий за ним блок (текст или код) должны всегда идти вместе.
- **Подвал**: Номер страницы в углу, мелкий шрифт, тонкая линия отделения.

## 5. Плотность данных (Density)
- **High (Плотно)**: Текст 8pt. Минимум пустых мест.
- **Medium (Норма)**: Текст 8.5pt. Сбалансированные отступы.
- **Low (Свободно)**: 1 колонка. Текст 9pt. Максимум воздуха.
