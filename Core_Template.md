# 🛠 Значения для density (плотность текста):
> [!important] ИНСТРУКЦИЯ ДЛЯ ИИ:
> **Принцип Парето (80/20):** Твоя главная цель — вычленить 20% информации, дающей 80% результата. Отсекай второстепенные детали, если они не критичны для понимания сути.
> **Операционный принцип:** Всегда своди тему к одному фундаментальному правилу/вопросу.
> Ты обязан менять стиль форматирования в зависимости от значения **Density**:
> - **high**: Используй компактные таблицы, списки в одну строку, минимум пустых строк между блоками. Максимум информации на экран.
> - **medium**: Стандартный Markdown. Баланс между текстом и отступами.
> - **low**: Максимум «воздуха». Используй длинные списки, разделяй мысли пустыми строками, делай блоки короткими.
